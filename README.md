## TROUBLESHOOTING

**Images not showing:**
- Check filenames match exactly (case-sensitive)
- Verify images are in root folder (not in subfolder)
- Clear browser cache
- Wait for GitHub Pages rebuild

**Fonts not updating:**
- Force refresh (Shift + Refresh)
- Clear browser cache
- Check that HTML files include Google Fonts link

**Site not updating:**
- Go to Settings → Pages
- Verify Source is "Deploy from a branch: main"
- Check Actions tab for build status
- Re-deploy if necessary
